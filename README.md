# NovaBoard

NovaBoard 是一个轻量级的个人启动面板，专为将常用的 AI 助手、OCR 工具和效率工具聚合到同一个入口而设计。它可以本地运行，不需要任何后端服务，通过浏览器即可完成收藏、搜索、自定义快捷入口等操作。

## ✨ 主要特性

- **分门别类的入口**：内置 OCR、AI、效率工具、个人中心等类别，核心入口一目了然。
- **收藏夹与搜索**：给常用功能加星标即可收纳到收藏夹，搜索框可实时过滤全部入口。
- **个性化面板**：自定义昵称、欢迎语与主题色，打造属于你的启动页。
- **自定义快捷入口**：可添加任意网页或内部工具为快捷入口，随时删除或调整。
- **本地数据保存**：所有偏好（收藏、自定义入口、表单数据等）均存储在浏览器 `localStorage` 中，无需登录账户。
- **内置效率小工具**：包含提示语手册、快写便笺等轻量工具，支撑日常工作流。

## 📁 目录结构

```
├── assets/            # 全局静态资源
│   ├── css/style.css  # 样式与主题
│   ├── icons.svg      # 内联图标库
│   └── js/app.js      # 公共脚本（入口渲染、收藏、自定义逻辑）
├── category/          # 各分类页面（由 app.js 动态填充）
├── tools/             # 具体工具页面（OCR、提示语手册等）
├── favorites.html     # 收藏夹聚合页
├── index.html         # 首页
├── profile.html       # 个人资料页
└── settings.html      # 偏好设置页
```

## 🚀 使用方式

1. 克隆或下载本仓库：

   ```bash
   git clone <repo-url>
   cd try
   ```

2. 直接通过浏览器打开 `index.html` 即可使用。为了获得更佳体验，建议使用本地静态服务器，例如：

   ```bash
   # 使用 Python 简易服务器
   python -m http.server 8000
   ```

   然后访问 `http://localhost:8000/try/`。

3. 所有数据都存储在浏览器端。若需要重置，可在浏览器开发者工具中清除对应站点的本地存储。

## 🛠️ 自定义入口指南

- 打开 **“设置”** → **“自定义快捷入口”**。
- 填写名称、描述与链接，可选择是否在新标签页打开。
- 提交后入口会出现在“全部入口”及搜索结果中，并可添加到收藏夹。

## 🧩 浏览器兼容性

- 推荐使用最新版本的 Chromium 内核或 Firefox 浏览器。
- 本项目依赖 ES6+ 原生特性（如 `const`、`classList`、`fetch` 等），无需额外 polyfill。
